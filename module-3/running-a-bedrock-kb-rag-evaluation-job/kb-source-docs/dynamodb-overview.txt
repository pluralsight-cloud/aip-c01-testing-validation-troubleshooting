Amazon DynamoDB Overview

Amazon DynamoDB is a fully managed NoSQL database service that provides fast and predictable performance with seamless scalability. DynamoDB supports both key-value and document data models. Tables in DynamoDB are schemaless, meaning each item can have a different set of attributes. DynamoDB uses primary keys to uniquely identify each item in a table. A primary key can be a simple partition key or a composite key consisting of a partition key and a sort key. DynamoDB supports two read consistency models: eventually consistent reads and strongly consistent reads. DynamoDB offers two capacity modes: on-demand and provisioned. DynamoDB Streams captures a time-ordered sequence of item-level modifications in a DynamoDB table. Global tables provide a fully managed solution for deploying a multi-Region, multi-active database. DynamoDB supports transactions with ACID properties across multiple items and tables.
