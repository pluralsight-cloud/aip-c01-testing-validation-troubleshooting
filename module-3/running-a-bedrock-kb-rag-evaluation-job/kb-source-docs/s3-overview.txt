Amazon S3 Overview

Amazon Simple Storage Service (Amazon S3) is an object storage service offering industry-leading scalability, data availability, security, and performance. S3 stores data as objects within buckets. An object consists of a file and optionally any metadata that describes that file. S3 offers multiple storage classes designed for different use cases: S3 Standard for frequently accessed data, S3 Intelligent-Tiering for data with unknown or changing access patterns, S3 Standard-IA and S3 One Zone-IA for infrequently accessed data, S3 Glacier Instant Retrieval, S3 Glacier Flexible Retrieval, and S3 Glacier Deep Archive for long-term archive. S3 provides 99.999999999% (11 nines) of data durability. S3 supports versioning to preserve, retrieve, and restore every version of every object stored in a bucket. S3 bucket names must be globally unique across all AWS accounts.
